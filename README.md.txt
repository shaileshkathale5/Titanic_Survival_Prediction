# Titanic Survival Prediction

This project predicts whether a passenger survived the Titanic disaster using Machine Learning.

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

## Machine Learning Model
- Logistic Regression

## Features
- Data Cleaning
- Data Visualization
- Survival Prediction
- Accuracy Analysis

## Accuracy
- 79% Accuracy
